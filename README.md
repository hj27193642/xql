# xql API (json to graphql)

Simple & Light headless CMS  using json


## Getting Started

### poetry install

```
RUN curl -sSL https://install.python-poetry.org | python3 -
```
Set path
```
PATH /home/homeuser/.local/bin:$PATH
```

packae update

```
cd /work
poetry update
```

run application server
```
poetry run python src/main.py
```
